# Diaspora Hub

Прототип веб-платформы для диаспоры, соответствующий ТЗ из `TZ_website_diaspora.md`.

## Что включено

- Лендинг с ключевыми разделами: функции, платежи, личный кабинет, админ-панель и контакты.
- Секции, отражающие бизнес-требования (оплата, отчеты, уведомления).
- Демонстрационные формы и карточки статусов.

## Локальный запуск

```bash
cd web
python -m http.server 8000
```

Откройте `http://localhost:8000` в браузере.
